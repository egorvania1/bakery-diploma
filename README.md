# Дипломный проект
Информационная система для удаленного заказа кондитерских изделий из сети кондитерских
## Как запустить
- Создать виртуальное окружение pip
- Установить требуемые пакеты в [requirements/release.txt](requirements/release.txt)
- Сделать миграции `python manage.py makemigrations`
- Провести миграции `python manage.py migrate`
- Запустить через `python server.py`
